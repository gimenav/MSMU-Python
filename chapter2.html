Chapter 2: Variables, Statments and Expressions

What is a value?

A value is a number or a letter that can be manipulated by a program.

What are data types?

Data types are the types of data that are processed in a program. Examples of data types range from numbers to sentences.

What is an assignment token?

An assignment token (=) is a token that assigns a value to a variable. This is not to be mistaken with the equal sign that is used in arithmetic computations.

What are operators and operands?

Operators are special symbols used for arithmetic or logical computations. Operators are used for addition, subtraction, multiplication and division. While operands are the values operators used for computations.

What is function?

A function is a block of code used to execute a certain action repeatedly, that is called by the name given.

What is an argument in Python

An argument is a value passed to a function or a method, when calling the function.

What does it mean to update a variable?

When updating a variable, first the current value of the variable has to be initialized(assign a value to the variable). Afterwards, the variable is either added (incremented), subtracted (decremented), divided or multiplied by another variable or value, lastly it is updated to the new value.
